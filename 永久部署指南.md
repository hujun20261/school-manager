## 🔑 准备工作（10 分钟）

### 1️⃣ 注册 Netlify 账号
1. 电脑打开 👉 https://app.netlify.com/signup
2. 用 **GitHub / Google / Email** 任一方式注册（推荐 GitHub，最快）
3. 注册完自动跳到后台首页

### 2️⃣ 准备网站文件
从 AI 助手处获得 `school-manager-mobile/dist` 文件夹（已打包为 ZIP）
→ 解压到任意目录，记住路径

---

## 🚀 部署（拖拽，30 秒）

1. 登录 Netlify 后台 👉 https://app.netlify.com
2. 点页面中间的 **「Add new site」** → 选 **「Deploy manually」**
3. **把 `dist` 文件夹直接拖进去**
4. 等进度条跑完（约 10-20 秒）
5. 屏幕显示绿色 ✅，**给你一个链接**，例如：
   ```
   https://splendid-cranberry-xxxxx.netlify.app
   ```
6. **点一下这个链接，确认能打开** → 成功！🎉

---

## 🔗 自定义网址（可选，5 分钟）

如果不喜欢随机网址：
1. Netlify 后台 → 点你的站点 → **「Site settings」**
2. 左边菜单点 **「Change site name」**
3. 输入喜欢的名字，比如：
   ```
   school-manager.netlify.app
   ```
4. 点保存 → 以后用这个新网址访问

## 📱 手机上使用

1. 用手机浏览器（Safari / Chrome）打开上面的永久链接
2. Safari：点底部分享按钮 → **「添加到主屏幕」**

## 🔄 数据说明

- 所有数据存在**浏览器本地**（localStorage）
- 清浏览器缓存会丢失数据 → 建议定期截图备份
- 不同电脑/手机打开是**独立数据**，互不影响

## ❓ 常见问题

**Q：网址永久有效吗？**
✅ 只要不删除 Netlify 站点就一直有效。

**Q：能离线用吗？**
第一次打开后，页面会缓存到本地，以后断网也能打开（但数据只存在本地）。

**Q：怎么改功能？**
联系 AI 助手，拿到新版 `dist` 文件夹，重新拖到 Netlify（覆盖部署）。
